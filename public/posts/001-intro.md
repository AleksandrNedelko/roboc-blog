2. Настроил переменные окружения
3. Создал первый проект "Hello World"
4. Научился компилировать программы через командную строку

## Пример кода

```c
#include <stdio.h>

int main() {
    printf("Hello, Robotics!\n");
    return 0;
}
```

# Что понял

- C - это компилируемый язык, программы нужно компилировать перед запуском
- `#include` используется для подключения библиотек
- `main()` - точка входа в программу
- `printf()` - функция для вывода текста

# Ошибки

При первой компиляции забыл точку с запятой в конце строки. Компилятор выдал ошибку синтаксиса, быстро исправил.
